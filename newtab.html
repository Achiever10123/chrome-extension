Help - SilentTab Extension

Welcome to SilentTab! Here's how to use its features:

1. Greeting & Clock

Displays a personalized greeting based on the time of day and a live clock.

2. Search Bar

Type your query and select a search engine (Google, DuckDuckGo, Bing) to search the web.

Tip: Press / on your keyboard to focus the search bar.

3. Shortcuts

Add frequently visited websites using the "Add Shortcut" form.

Hover over a shortcut tile to reveal the delete button (×).

You can add shortcuts by name and URL. The extension will try to fetch the website's favicon.

Tip: Press Ctrl+S (or Cmd+S on Mac) to focus the shortcut name field.

4. To-Do List

Add tasks using the "Add" button. Click on a task to mark it as completed (strikethrough).

Hover over a task to reveal the delete button (×).

Tip: Press Ctrl+T (or Cmd+T on Mac) to focus the to-do input field.

5. Settings

Click the ⚙️ Settings button to customize:

  • Your name (for the greeting)
  • Default search engine
  • Theme (Liquid Glass, Aqua, Light, Dark)
  • Background (Default or Custom Image URL)

6. Stats

Displays the number of shortcuts added and the number of completed/total to-dos.

7. Keyboard Shortcuts

  • / - Focus search bar
  • Ctrl+T (or Cmd+T) - Focus to-do input
  • Ctrl+S (or Cmd+S) - Focus shortcut name input
